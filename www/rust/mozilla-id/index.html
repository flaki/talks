What is Rust?

  • The future of systems programming
  • "…a systems programming language that runs blazingly fast,
    prevents segfaults, and guarantees thread safety."

This wonderful "Alien Technology"

  • Rust is already over 10 years old!
  • Developed first as a side project (as all world-changing projects usually are)
  • Created by Graydon Hoare
  • Rust 1.0 was released on May 15, 2015

A browser's biggest fear: Security Vulnerabilities

  • Pwn2Own is a competition
  • Most vulnerabilities in browsers to date still stem from code errors (off-by-one errors, use-after-free bugs)

Rust’s Fix

  • compile-time integrity checking
  • compile-time abstraction unwrapping
  • High level–but in a low level way
  • zero cost abstractions
  • Restricting unsafe code

Fearless concurrency

  • Concurrency in software is HARD
  • no shared mutable state
