Why Firefox OS?

Firefox OS development
=

cutting edge web development.

 

  • Next-generation JavaScript features (ECMAScript 6+)
  • Experimental APIs for platform-parity
  • Open Web APIs for accessing hardware & device resources

ECMAScript & TC39

ECMAScript's evolution is a joint effort, led by the TC39 commitee of browser vendors and other industry experts.

  • ES6 Draft expected to finalize before the end of year
    (ECMA approval target: December, 2014)
  • ECMAScript rapid iteration (proposed) roadmap:
    ES6/7/8 and onward
  • New ECMAScript versions will come in "rapid" iterations (approx. one a new version every year)
    Active ES7 Proposals

If you have an index.html, you have (almost) got a Firefox OS app!
